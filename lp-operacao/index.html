4

Operação (a partir do mês 2)

Volume total ligado. Manutenção mensal: monitoro custo de IA, ajusto prompts conforme novos produtos, plugo integrações novas. Você acompanha tudo pelo CRM.

Perguntas frequentes

Mesma persona, três conjuntos de regras. O agente identifica em qual estágio o contato está (lead novo? lead qualificado? cliente pagante?) consultando o CRM antes de cada resposta. Pra lead novo ele puxa o playbook de SDR. Pra lead quente puxa SPIN Sell. Pra cliente puxa base de conhecimento do produto. O usuário só vê uma pessoa — Camila/Luiza/seu nome — falando com ele.

Em produção e os leads não percebem. Tem 9 camadas de guardrail que bloqueiam qualquer admissão de "sou IA/bot/assistente". O agente também tem delays humanos (3-12s entre mensagens), responde áudio com áudio quando faz sentido, e usa o vocabulário da sua marca.

Nunca aconteceu nos projetos em produção. Uso Evolution API para WhatsApp (rotação de sessão, delay humano, rate limit 60 msg/hora) e Instagram Graph API oficial com permissões aprovadas pela Meta. Pra Instagram, levo seu app para review oficial — você fica em conformidade total.

Arquitetura de 3 camadas: diretivas determinísticas em Markdown → IA orquestra → scripts Python executam. A IA não envia link de checkout direto, ela chama uma tool que envia o link correto. Pra Suporte, ela só responde sobre o que está na base de conhecimento — fora disso, escala pro humano.
